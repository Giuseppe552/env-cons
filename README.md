# Food Science & Environmental Health Consulting

A minimalist single-page Next.js website for a food science and environmental health consultant.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Structure

- Single-page application with sections for:
  - Hero with headline and CTA
  - Services
  - About
  - Contact

## Deployment

This site is configured for deployment on Vercel through GitHub.

## Content

Content placeholders are marked with `[to be filled in]` - update these in:
- `site.json` for basic site configuration
- `app/page.tsx` for page content
- `app/layout.tsx` for metadata
